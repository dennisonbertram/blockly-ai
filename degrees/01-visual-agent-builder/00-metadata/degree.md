# Visual Agent Builder — Blockly + Vercel AI SDK

## Audience

LLM agents that will build similar visual-agent-builder projects. Every artifact in this degree is written so a downstream model can read it, follow citations, and reproduce or extend the work without needing access to the original human authors.

## Prerequisites

- Working knowledge of TypeScript (types, generics, strict mode).
- Basic understanding of LLM concepts: prompting, tool calling, structured output, streaming.
- React and Next.js (App Router) basics — enough to read and write a route handler and a client component.

## Deliverables

- **POCs L1–L5** plus a capstone, each with source, tests, and an evidence log.
- **Distillation artifacts** — gotchas, patterns, anti-patterns, playbooks, and a "before-you-build" checklist.
- **A navigable skill pack** under `06-skill-pack/` (lessons, labs, recipes, troubleshooting) suitable for direct consumption by an LLM agent.
- **A deployable capstone**: a research-and-summarize agent constructed visually in Blockly, compiled to AI SDK code, and deployed to Vercel.
